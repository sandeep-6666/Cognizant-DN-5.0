## How to Run

1. Clone the repository:

```bash
git clone https://github.com/sandeep-6666/Cognizant-DN-5.0.git
```

2. Navigate to the required project folder.

3. Build the project:

```bash
mvn clean install
```

4. Run the project using:

```bash
mvn spring-boot:run
```

or run the Java class containing the `main()` method.
